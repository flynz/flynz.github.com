    新西兰工作转居留签证简介 - FLYabroad

    新西兰工作转居留签证 Residence from Work visa 适用于已经在新西兰境内持有 Work to Residence visa，打算申请居留签证的申请人。

    此签证只能在新西兰境内申请。伴侣和子女可以随同，身体健康，无犯罪记录，配偶和大于16岁的子女，雅思不低于平均分5。

    新西兰工作转居留签证申请基本要求 - FLYabroad

    1. 身体健康
    2. 无犯罪记录
    3. 长期短缺职业类 - Long Term Skill Shortage List Work category、杰出人才（认可雇主类） - Talent (Accredited Employers) Work category、杰出人才（艺术、文化或体育）- Talent (Arts, Culture and Sports) Work category 主申请人无需满足英语要求，但是宗教工作主申请人需要雅思不低于平均分5

    新西兰工作转居留签证分类介绍 - FLYabroad

    根据申请人持有的工作期满专为居留类临时签证 Work to Residence visa 的不同，对申请人的要求也不同。

    1. 持有杰出人才（认可雇主类） - Talent (Accredited Employers) Work category 申请人的要求

    持有此签证在新西兰境内为认可的雇主工作2年以上；
    当前仍在工作，且年薪在55,000新西兰元以上；
    需要满足职业注册要求（如果职业要求）；
    全职（每周30小时以上）工作。

    2. 持有杰出人才（艺术、文化或体育）- Talent (Arts, Culture and Sports) Work category 的申请人的要求

    在新西兰境内持有此签证活跃的从事该领域工作2年以上；
    在所从事领域成绩仍旧突出；
    从未申请过任何新西兰社会福利；
    获得新西兰国内有一定声望的机构的担保。

    3. 持有长期短缺职业类 - Long Term Skill Shortage List Work category 申请人的要求

    持有此签证在新西兰境内工作2年以上；
    当前仍在工作，且年薪在45,000新西兰元以上；
    55岁以下（含55岁）；
    需要满足职业注册要求（如果职业要求）。

    4. 持有宗教工作者工作签证的申请人的要求

    持有宗教工作者工作签证3年以上；
    有慈善机构为你提供担保；
    担保机提供的真实、不间断的工作offer；
    5年以上宗教培训或宗教工作经验；
    55岁以下（含55岁）；
    雅思不低于平均5；
    从未申请或得到过新西兰社会福利。
